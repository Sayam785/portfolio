Projects

University Timetable Management (C)

Console program • C, File Handling

A straightforward C program that helps generate and view class timetables. It stores inputs (courses, faculty, rooms, time slots) in files, checks for basic clashes, and prints readable views (day-wise or faculty-wise). The focus was on clear menus, input validation, and keeping the code modular for each step (input, validate, schedule, view, update).

  • Modules: Input, Validation, Scheduling, View/Report, Update, File I/O, Main Menu.
  • Outputs: human-readable table prints; faculty/day wise reports.
  • Data storage: basic flat files with simple formats for quick loading/saving.

Parking Management (C)

Console program • C, File Handling

A small file-based project to manage parking slots. It notes vehicle entry/exit, tracks free slots, and prints basic bills based on duration. The point was to practice simple record storage and retrieval with file streams and keep the code easy to navigate.

  • Features: entry/exit logs, availability, simple billing.
  • Emphasis on: structured menus, safe input, and persistent storage.
